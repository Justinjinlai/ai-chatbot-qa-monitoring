# AI Chatbot QA & Monitoring System

This project simulates AI Operations work by analyzing chatbot conversations
to identify response quality, potential failures, and customer sentiment.

## Features
- QA scoring of chatbot responses
- Identification of responses requiring review
- Basic sentiment analysis
- Operational insights for improvement

## Tech Stack
- Python
- CSV-based log simulation

## How It Works
1. Chatbot conversations are logged in a CSV file
2. Responses are scored using QA rules
3. Sentiment analysis provides operational insights
4. Results help identify areas for AI improvement

## Why This Matters
AI systems require continuous monitoring after deployment.
This project demonstrates skills in:
- Quality assurance
- Log analysis
- AI operations monitoring
- Customer experience optimization
